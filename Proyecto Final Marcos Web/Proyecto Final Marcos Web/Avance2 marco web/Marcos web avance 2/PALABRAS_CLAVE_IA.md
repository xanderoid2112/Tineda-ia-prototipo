# 🎯 Palabras Clave que Entiende la IA

## **MARCAS** (Productos preferidos)
- **Peruanas**: Gloria, Nestlé, Bimbo, Laive, Primor, Costa, Costeño, Donofrio, Molitalia, San Fernando, Altomayo, Inca Kola, San Luis, Field, Bells, Molicentro, Pura Vida
- **Internacionales**: Coca-Cola, P&G, Unilever, Kraft, General Mills, Chips, Oreo, Doritos, ACE, Rexona, Scott

**Cómo usar**: Click en los botones de marcas O escribir en preferencias: `marcas Gloria y Bimbo`

---

## **DIETAS** (Reconocidas)
- `vegana` / `vegano` / `veganos` / `vegan` → **vegana**
- `vegetariana` / `vegetariano` / `vegetarianos` → **vegetariana**
- `sin gluten` / `gluten free` → **vegetariana**
- `sin lactosa` / `lactose free` → **vegetariana**
- `orgánico` / `orgánica` / `organic` → **regular**
- `baja en calorías` → **regular**
- `sin azúcar` → **regular**

**Cómo usar**: Escribir en preferencias: `dieta vegana`

---

## **CATEGORÍAS A EXCLUIR**

### Carnes y Productos Animales
- `carne` → excluye **Carnes**
- `carnes` → excluye **Carnes**
- `pollo` / `pollos` → excluye **Carnes**
- `cerdo` → excluye **Carnes**
- `res` → excluye **Carnes**

### Pescados y Mariscos
- `pescado` / `pescados` → excluye **Pescados**
- `mariscos` → excluye **Pescados**

### Bebidas (Alcohol y Refrescos)
- `alcohol` → excluye **Bebidas**
- `licor` → excluye **Bebidas**
- `cerveza` → excluye **Bebidas**
- `vino` → excluye **Bebidas**
- `bebidas alcohólicas` → excluye **Bebidas**
- `gaseosas` / `refrescos` → excluye **Bebidas**

### Snacks y Dulces
- `snacks` / `chips` / `papas fritas` → excluye **Snacks**
- `dulces` / `chocolate` / `golosinas` → excluye **Dulces**

**Cómo usar**: Escribir en preferencias: `excluir carnes, alcohol, pescados`

---

## **EJEMPLOS DE USO**

### Ejemplo 1: Solo marcas
```
marcas Gloria y Bimbo
```
→ Prioriza productos de Gloria y Bimbo

---

### Ejemplo 2: Solo dieta
```
dieta vegana
```
→ Solo productos veganos

---

### Ejemplo 3: Exclusión
```
excluir carnes, alcohol
```
→ No incluye carnes ni bebidas alcohólicas

---

### Ejemplo 4: Completo
```
dieta vegetariana, marcas Laive y Pura Vida, excluir pescados y gaseosas
```
→ Productos vegetarianos de esas marcas, sin pescados ni gaseosas

---

### Ejemplo 5: Súper específico
```
no quiero pollos, cerdo, mariscos, alcohol
```
→ Excluye carnes, pescados y alcohol

---

## **COMBINACIONES POSIBLES**

**MARCAS + DIETA**
```
marcas Gloria, dieta sin lactosa
```

 **DIETA + EXCLUSIÓN**
```
dieta vegana, excluir snacks
```

**MARCAS + EXCLUSIÓN**
```
marcas Bimbo y Laive, excluir alcohol
```

**TODAS**
```
marcas Gloria, Bimbo, dieta vegetariana, excluir pescados y alcohol
```

---

##  **IMPORTANTE**

1. **"excluir" es la palabra clave para exclusiones**
2. No es necesario usar "excluir" si solo usas marcas o dietas
3. Las marcas se pueden seleccionar con los botones O escribirlas en texto
4. El sistema es **inteligente** y entiende variaciones de palabras

---

##  **CATEGORÍAS DE PRODUCTOS**

El sistema reconoce estas categorías para exclusiones:
-  **Bebidas** (agua, gaseosas, alcohol)
- **Carnes** (pollos, res, cerdo)
-**Pescados** (pescados, mariscos)
-  **Conservas** (lata, enlatados)
-  **Lácteos** (leche, yogurt)
-  **Granos** (arroz, fideos)
-  **Frutas** (frutas frescas)
-  **Snacks** (chips, papas)
-  **Dulces** (chocolate, golosinas)
